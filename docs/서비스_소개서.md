# Excel Broadcast Settlement 서비스 소개서

## 서비스 한 줄 소개

**방송 후원을 실시간으로 관리하고, 방송이 끝나면 한 번에 정산까지 처리하는 올인원 방송 관리 시스템**입니다.

---

## 1. 이 서비스는 누구를 위한 것인가요?

| 대상 | 활용 방식 |
|------|------------|
| **유튜브/트위치 방송자** | 후원(계좌이체·투네이션) 실시간 집계, 방송 중 시청자에게 진행 상황 표시 |
| **방송 팀 운영자** | 멤버별 후원 분배, 정산서 자동 생성, PDF/CSV 내보내기 |
| **소규모 방송팀** | 별도 정산 인력 없이, 방송 종료 후 바로 정산 완료 |

---

## 2. 왜 이 서비스를 사용해야 할까요?

### 2.1 방송 중

- **실시간 후원 집계**: 계좌이체·투네이션 금액을 즉시 반영
- **화면 오버레이**: OBS·Prism에 멤버별 금액, 총합, 목표 달성률 등을 바로 표시
- **미션 전광판**: 후원 미션을 우측→좌측 흐르는 전광판으로 노출
- **개인 목표**: 멤버별 목표 금액과 진행률을 한눈에 확인

### 2.2 방송 종료 후

- **자동 정산**: 계좌·투네 비율, 세금 비율을 적용해 멤버별 정산금 계산
- **다양한 형식 내보내기**: TXT, CSV, PDF로 저장·공유
- **정산 기록 관리**: 과거 정산 내역 검색 및 재조회

### 2.3 운영 편의

- **웹 기반**: 별도 설치 없이 브라우저에서 사용
- **모바일 앱**: Android 앱으로 외부에서도 확인·다운로드 가능
- **다중 계정**: 팀별·프로젝트별로 계정 분리 운영

---

## 3. 주요 기능 요약

### 3.1 관리자 대시보드

| 기능 | 설명 |
|------|------|
| **멤버 관리** | 멤버 추가/삭제, 계좌·투네 금액 입력, 개인 목표 설정 |
| **후원자 기록** | 후원자명, 금액, 대상 멤버, 건별 기록 관리 |
| **미션 관리** | 후원 미션 추가·수정·삭제, HOT 표시 |
| **오버레이 설정** | 여러 화면 구성을 프리셋으로 저장, OBS/Prism용 URL 생성 |
| **정산 생성** | 비율 설정 후 정산 계산, TXT/CSV/PDF 내보내기 |

### 3.2 방송 오버레이

| 표시 요소 | 설명 |
|-----------|------|
| **멤버 목록** | 순위, 이름, 계좌, 투네, 합계 (표 형태) |
| **총합** | 전체 계좌·투네·총액 |
| **목표바** | 전체 목표 금액 대비 진행률 |
| **개인 골** | 멤버별 목표 달성률 |
| **후원 티커** | 최근 후원자 스크롤 |
| **타이머** | 방송 경과 시간 |
| **미션 전광판** | 미션 목록 우측→좌측 흐르는 전광판 |

### 3.3 테마

- **기본·엑셀·네온·레트로·미니멀·RPG·파스텔** 등 다양한 스타일
- **그라데이션 테마**: 무지개, 일몰, 오션, 포레스트, 오로라, 바이올렛, 코랄, 민트, 라바, 아이스
- 테마 하나로 멤버 목록·총합·목표바·티커 등 전체 스타일 통일

## 4. 사용 흐름 (예시)

```
1. 로그인
   └─ 관리자 페이지 접속

2. 방송 전 준비
   ├─ 멤버 추가 및 개인 목표 설정
   ├─ 미션 추가 (선택)
   └─ 오버레이 프리셋 설정 후 OBS/Prism에 URL 등록

3. 방송 중
   ├─ 후원 발생 시 금액 입력 (계좌/투네, 대상 멤버)
   ├─ 오버레이가 실시간으로 갱신
   └─ 시청자에게 진행 상황 표시

4. 방송 종료
   ├─ 정산 비율 설정 (계좌·투네·세금)
   ├─ 정산 생성
   └─ TXT/CSV/PDF로 내보내기
```
